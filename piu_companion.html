PIU Companion App

The app is designed to help students and lecturers obtain information and navigate through campus with ease.

The features of this app includes:

  1. Real-time class assignments - In this feature, when a lecturer posts a class assignment you will get it instantly. Kindly note that this applies only for the units you have registered for.
  2. Attendance generator and scanner with cloud storage - This feature generates a QR code based on the encrypted data of the user.
    Note that it will only generate a code when the class session is opened and the only device that can mark your attendance is the lecturers device. All you need is to input the unit your are attending and the Lecturer will scan and it will mark your attendance automatically.
  3. Real-time class announcements - This feature displays the class announcements of the units you have registered, this class announcements will help students know in advance when a class has been postponed or to note an upcoming Continuous Assessment Test.
  4. Power Options Control - Sometimes we need to reset our console, you can now do that from your palm of your hand and a push of a button, you have three options to choose from, Hard Reboot, Soft Reboot(Less aggressive) and Shutdown.
  5. Google Analytics for user behavior statistics - This is to help the lecturers and developer to get reports on the usage of the app.

    This will provide data from daily users to the region the user is logged in
